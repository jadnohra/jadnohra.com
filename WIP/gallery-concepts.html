Recommendation

Best fit for your portfolio: Concept 2 (Collapsible) + Concept 1 (Inline) hybrid

Why:
  • Works with Jekyll/Chirpy markdown using native HTML
  • 1-2 small inline thumbnails for immediate context
  • <details> tag for full gallery when available
  • Keeps page scannable while offering depth
  • No JavaScript required
