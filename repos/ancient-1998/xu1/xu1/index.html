3D Racing (2001)
XU1 Productions was a group of around 10 people trying to make a 3D Racing game during their free time.
The project died off and doesnt exist anymore.
As a programmer I wrote the following components:
  • Fast Terrain Pre-processing and rendering engine using an Octree subdivision.
  • 3D Studio .ase file import.
  • Resource Management Libraries.
  • Sound Engine able to handle mp3 files, Audio CD, wave files, 3D sound.
  • Event based GUI engine
