'Yet Another' Space Shooter Applet (2001)
Unfortunately I never got any art done so no explosions, no animations,
no transparency, although in the code they are supported.
The Nice Working Feature of .xml file configuration works fine, have a look
at the current config. HERE.. (Right Click, Save Target As...)
Some Features:
  • .xml configuration
  • overriden java's resource loading to allow exact monitoring
    of loaded byte count
  • Simple GUI
  • Extra Accurate Collision Detection: Objects do no 'pass through'
    each other no matter how fast they are
Applet In Action
Select A ship with the mouse, click inside the applet.
keys:
  • Num. Pad for ship movement, SPACE to shoot
  • 'E' to create a new Enemy!!! :))))
  • Player ship takes no damage :))))
  • Reload Page to select other ship.
