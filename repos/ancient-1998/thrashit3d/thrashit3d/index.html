'Thrashit 3D' 3D Software Renderer (2001)
One of my Early efforts to learn about the internals of 3D rendering By writing a Software Renderer.
The Following Was Implemented:
  • Z Buffering
  • WireFrame, Flat and Gouraud Shading
  • Rasterization with Wire Frame Mode support
  • Full Screen / Windowed
  • Log File + Performance Measurment
